# Retailaer VSCode Theme

A professional color theme for Visual Studio Code based on Retailaer's brand identity.

## Color Palette

The theme uses Retailaer's official design system colors:

- **Primary Teal**: `#0a5c5c` - Deep, professional teal representing trust and travel technology
- **Accent Yellow**: `#f5b800` - Bright yellow for highlights, CTAs, and important UI elements
- **Background Light**: `#f0f7f7` - Soft teal-tinted background
- **Text Dark**: `#0a1f1f` - Primary text color
- **Text Body**: `#5a7a7a` - Secondary text color

## Variants

### Retailaer Dark
A dark theme with deep teal backgrounds and bright yellow accents. Perfect for long coding sessions with reduced eye strain.

### Retailaer Light
A clean light theme with white backgrounds and teal accents. Great for daytime work and presentations.

## Features

- Carefully selected colors that match Retailaer's brand
- Both dark and light variants
- Optimized syntax highlighting for popular languages
- Enhanced UI elements (sidebar, tabs, status bar)
- Git integration colors
- Terminal color scheme
- Accessible contrast ratios
